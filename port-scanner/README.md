# Port Scanner

Advanced network port scanner with CLI and web GUI.

## Files
- `portscanner.py` — Terminal CLI version
- `portscanner_gui.py` — Browser-based GUI (opens http://localhost:7331)

## Usage
```bash
# GUI
python3 portscanner_gui.py

# CLI interactive
python3 portscanner.py

# CLI quick
python3 portscanner.py scanme.nmap.org --preset web
```

## Requirements
Python 3.8+ · No external dependencies
